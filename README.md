# Andre's Web Portfolio

## Description
This is a personal web portfolio for Andre Nyanjahia, a Full-Stack Web & Systems Developer specializing in the Next.js and PERN stack. The portfolio showcases skills, projects, and contact information in a clean, responsive design.

## Features
- **Introduction Section**: Personal introduction with links to GitHub and LinkedIn.
- **Skills Table**: Detailed breakdown of competencies in System Architecture, Frontend Ecosystem, Backend & Databases, Data Strategy & Analytics, DevOps & Tools, and Design Tools.
- **Projects Showcase**: Highlighted projects including Brogram, Smolstore, Expenny, and various Saiket Systems assignments.
- **Interactive Color Picker Game**: A fun feature allowing users to change the background color.
- **Responsive Design**: Optimized for various screen sizes using custom CSS.

## Technologies Used
- **HTML5**: Structure of the web pages.
- **CSS3**: Styling, including custom stylesheets (`styles.css`, `fanta.css`).
- **JavaScript**: Interactive elements, such as the color picker game.
- **Font Awesome**: Icons for social media links and navigation.
